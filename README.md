## About the Project  
FitLife Glow is a clean, modern responsive website designed around fitness, wellness and health-lifestyle branding. Whether you’re offering workout programs, personal training, yoga flows or nutrition plans, this template provides a polished front-end foundation.

**Live Demo:** [https://rudra-ai-2127.github.io/fitlife-glow/](https://rudra-ai-2127.github.io/fitlife-glow/)  
**Repository:** [https://github.com/Rudra-AI-2127/fitlife-glow](https://github.com/Rudra-AI-2127/fitlife-glow)  

## Features  
- Fully **responsive layout** (desktop, tablet, mobile)  
- Hero banner with CTAs (join now, get started)  
- Section for services/programs  
- Section for trainer profiles or testimonials  
- Clean, modern UI/UX with smooth scroll and transitions  
- Built with modern front-end tooling (see tech stack below)  
- Ready for deployment (e.g., via GitHub Pages or Netlify)

## Tech Stack  
- **TypeScript** (primary codebase)  
- **CSS / Tailwind CSS** (styling)  
- **Vite** (build tooling)  
- **PostCSS** (CSS transformations)  
- Deployed via static host (Netlify or GitHub Pages)  
- Git + GitHub for version control
